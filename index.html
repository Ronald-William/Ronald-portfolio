PROJECTS

Library Management System

A Python-based library management system with a Tkinter GUI. It supports user roles for administrators (managing books and members) and library users (searching, borrowing, and returning books). Data is stored using MySQL and backed up in binary files. Features include book search, sorting, and exception handling (e.g., overdue books). The system enhances library efficiency by automating manual processes.

Job Portal

A responsive web platform that connects job seekers and recruiters. It allows recruiters to post jobs and manage applicants, while job seekers can search for jobs, apply, and upload resumes. The platform features advanced search filters, secure login, and a user-friendly interface optimized for multiple devices.
